## 🔬 Featured Projects

### 📡 AXI Protocol Validation using SystemVerilog & UVM
* Developed a comprehensive UVM environment to validate complex AXI protocol features and transaction scenarios.
* Implemented driver, monitor, agent, sequencer, scoreboard, and SystemVerilog Assertions (SVA) for protocol compliance.
* Verified advanced features including multiple burst types (incrementing, wrapping, fixed), aligned/unaligned transfers, overlapping, and out-of-order transactions.
* Performed log analysis and waveform debugging using Synopsys Verdi on Linux server environments.

### 🔄 Asynchronous FIFO Design & Verification
* Verified an Asynchronous FIFO design enabling data transfer across independent write/read clock domains.
* Verified pointer synchronization and Clock Domain Crossing (CDC) behavior.
* **Repository:** [View FIFO Project on GitHub](https://github.com/Aditya-kori/Asynchronous-FIFO-Pre)

### 💾 UVM-Based Memory Design Verification
* Designed Memory RTL in Verilog and verified it using SystemVerilog/UVM methodology.
* Created reusable UVM components (driver, sequencer, monitor, scoreboard) and used functional coverage (covergroups) to ensure full operational correctness.
* **Repository:** [View Memory Project on GitHub](https://github.com/Aditya-kori/Memory-Project-Pre)

### 🚣 IoT-Based RC Water Rescue Boat
* Developed an IoT-based remote-controlled rescue boat using NodeMCU and Arduino IoT Cloud for real-time dashboard telemetry.
* **Repository:** [View Water Rescue Boat on GitHub](https://github.com/Aditya-kori/waterBoat_dec13a)
